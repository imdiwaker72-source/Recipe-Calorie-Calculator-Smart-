# 🍽️ Smart Recipe Calorie Calculator

A simple, fast, and embeddable calorie calculator that converts plain text recipes into calorie and macro breakdown.

## 🚀 Live Demo
👉 https://imdiwaker72-source.github.io/Recipe-Calorie-Calculator-Smart-/
---

## 💡 Features

- Text-based input (no dropdowns)
- Quantity detection (e.g., "2 eggs")
- Instant calorie + macro calculation
- Lightweight & fast
- Fully embeddable

---

## 🧪 Example Input

2 eggs, 1 toast, 1 cup milk

---

## 📊 Output

- Calories
- Protein
- Carbs
- Fat
